# Antigravity

openagentsbtw installs Antigravity support only when Antigravity or Gemini-style config markers are detected.

## Surfaces

- Managed root `AGENTS.md` block
- Managed root `GEMINI.md` block

## Install

`./install.sh --antigravity` installs Antigravity support when Antigravity exists on the system.

`./install.sh --all` auto-detects Antigravity and skips it when missing.

## Hook Status

UNKNOWN official hook surface. openagentsbtw does not generate Antigravity hooks until official docs define a stable hook contract.
