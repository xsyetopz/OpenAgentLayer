# Concepts

OpenAgentLayer (OAL) makes existing coding agents stricter and more useful by adding source specs, native adapters, hooks, a runner, and validation.

Canonical source lives in `source/`:

- `product.toml`
- `models.toml`
- `runner.toml`
- `integrations.toml`
- `agents.toml`
- `skills.toml`
- `commands.toml`
- `hooks.toml`
- `platforms/*.toml`

See `docs/concepts/harness.md` for the harness boundary.
