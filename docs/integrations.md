# Integrations

OAL integrates with external tools by probing capabilities. It does not vendor or reinvent them.

- RTK: command-output filtering and token savings.
- Context7: third-party docs lookup with citations.
- Caveman: terse response mode pack.

See `docs/integrations/rtk.md`, `docs/integrations/context7.md`, and `docs/integrations/caveman.md`.
