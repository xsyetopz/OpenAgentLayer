# Install

v4 install is owned by `oal install <platform>`.

The beta target is:

```bash
oal install codex
oal install opencode
```

Install must write a manifest and remove only manifest-owned files during uninstall.

OS support:

- macOS
- Linux
- Windows through WSL2 only

See `docs/install/macos.md`, `docs/install/linux.md`, and `docs/install/wsl2.md`.
