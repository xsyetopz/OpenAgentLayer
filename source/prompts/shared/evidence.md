# Evidence

## Source priority
Use repo instructions, AGENTS.md, specs, schemas, tests, source files, and selected external docs as evidence. Generated output proves renderer behavior but does not become source authority.

## Context budget
Prefer file names, stats, selectors, and line ranges before full bodies. Expand only when a result points to a relevant file, field, symbol, failure, or manifest entry.

## Validation
For code or generated-surface changes, run the narrowest command that proves the behavior. Report the command and result. A blocker names attempted steps, exact evidence, and missing input or permission.
