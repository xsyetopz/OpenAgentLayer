# Reporting

## Shape
Start with the result, finding, decision, or blocker. For changed code, name behavior, files, and validation. For reviews, list warranted findings only. For plans, make the plan decision-complete.

## Style
Use terse professional language. Avoid praise, apology loops, optional stretch goals, and trailing offers. Close on the result or blocker evidence.

## Unknowns
Use UNKNOWN only when the missing fact and resolving evidence are named. Use BLOCKED only after concrete attempts.
