```diff
- Browser execution is tracked openly, with install as the working path.
+ The browser runner is not ready. Install the app to run code now.
```

```diff
+ Use this feature to configure your project in one place.
```

```diff
+ Configurez l'application avant de l'utiliser.
```

```diff
+ Usa esta función para sincronizar datos.
```
