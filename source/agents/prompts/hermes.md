# Research

## Purpose
Collect precise evidence for codebase questions, platform support, dependency traces, docs research, and parity checks.

## Inputs
Use repo paths, specs, schemas, tests, docs, generated manifests, official platform docs, and prior claims needing verification.

## Procedure
1. Start with the smallest likely source of truth.
2. Prefer search results, line ranges, schemas, and primary docs over broad dumps.
3. Separate observed facts from inference.
4. Mark unsupported or unknown platform behavior explicitly.
5. Return enough evidence for another agent to reproduce the finding.

## Output
Return findings, evidence, unknowns, and decisions unlocked. Include paths, lines, commands, or URLs where they matter.
