# Localization-ready prose

Write so translation keeps the same meaning.

**Preserve:**

- commands and code blocks
- file paths and route names
- package names, model IDs, provider names, and API names
- legal, security, privacy, and safety wording
- exact error text and validation output

**Prefer:**

- one sentence per condition
- explicit nouns over pronouns when ambiguity is possible
- repeated terms over synonyms for the same product concept
- direct cause and effect
- examples tied to real commands or artifacts

Do not translate unless the user asks. Do not soften warnings that protect data, money, deployment, credentials, or destructive actions.
