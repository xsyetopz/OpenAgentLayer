# Plain language and localization

Write for translation:

- one idea per sentence
- concrete subjects and verbs
- stable product terms
- active voice where it clarifies responsibility
- examples tied to real commands or UI labels

Preserve:

- command names
- flags
- file paths
- IDs
- API names
- legal/security terms
- validation evidence

Translation happens on user request. Security, money, deployment, credential, and destructive-action warnings keep their protective meaning.
