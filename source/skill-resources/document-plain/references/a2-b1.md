# A2-B1 plain English

Use this when the audience includes busy readers, non-native English readers, or people translating the text mentally.

**Rules:**

- Use short sentences.
- Put one idea in each sentence.
- Prefer active voice.
- Use concrete verbs.
- Put the user action or visible result first.
- Use the same word for the same idea.
- Keep headings direct and specific.
- Replace hype with observable behavior.
- Explain uncommon terms only when the reader needs them to act.

**Avoid:**

- robust, seamless, powerful, comprehensive, cutting-edge.
- perhaps, generally, usually, simply, arguably.
- long noun stacks.
- idioms, jokes, and culture-bound phrases.
- marketing rhythm that repeats three vague benefits.
