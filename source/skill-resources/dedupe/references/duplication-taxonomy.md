# Duplication Taxonomy

- Literal duplication: identical strings, URLs, path fragments, shell commands.
- Pattern duplication: repeated regexes or tokenization rules.
- Structural duplication: repeated object/array shapes and mapping logic.
- Behavioral duplication: same branching/validation logic in multiple files.
- Contract duplication: repeated schema markers, config keys, and prompt blocks.

Prioritize by risk: contract and behavioral duplicates first, then literals.
