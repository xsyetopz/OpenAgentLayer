# A2-B1 plain English

Use this when the audience includes busy readers, non-native English readers, or people translating the text mentally.

**Rules:**

- Use short sentences.
- Put one idea in each sentence.
- Prefer active voice.
- Use concrete verbs.
- Put the user action or visible result first.
- Use the same word for the same idea.
- Keep headings direct and specific.
- Replace hype with observable behavior.
- Explain uncommon terms when the reader needs them to act.

**Replacement targets:**

- robust, seamless, powerful, comprehensive, cutting-edge become observable behavior.
- perhaps, generally, usually, simply, arguably become exact scope or disappear.
- long noun stacks become short clauses.
- idioms, jokes, and culture-bound phrases become plain wording.
- marketing rhythm becomes concrete user outcomes.
