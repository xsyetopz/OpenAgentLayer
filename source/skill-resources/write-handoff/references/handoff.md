# Handoff content

Include:

- objective and current status
- Continuation Record: objective, done, next, blockers
- changed files with why
- commands run with exit codes
- observed symptoms and root cause evidence
- files intentionally left unchanged
- remaining decisions
- next safe action

Acceptance is recorded as passed when the acceptance command completed successfully in this session.

Keep the record short enough to paste as a user message after compaction or session transfer. Current user messages and verified repo evidence define the active path; markdown memory is supporting evidence after it is read.
