# Naming

Purpose: v4 naming spec.

Authority: normative.

## Product

- Full name: OpenAgentLayer.
- Short name: OAL.
- Category: portable agent behavior layer.

## Package and CLI

- Package root: package-module workspace under `packages/*`.
- Package names: `@openagentlayer/*`.
- There is no monolithic `packages/oal` package.
- CLI binary: `oal`.
- Source namespace: `openagentlayer`.
- Generated bundle metadata key: `openagentlayer`.

## Terms

- Surface: target agentic tool or IDE.
- Adapter: renderer/installer bridge for one surface.
- Source graph: normalized canonical records.
- Policy: declared behavior rule.
- Runtime guard: executable policy enforcement.
- Route contract: completion and work-shape requirement.
